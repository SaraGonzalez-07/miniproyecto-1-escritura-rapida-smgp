## Project Description
"Escritura Rápida" is a typing speed game developed using Java and JavaFX.
The player must type the word displayed on the screen before the time runs out.

The game is divided into levels, where each level increases the difficulty by 
reducing the available time and using more complex words. The objective is to 
reach the highest level possible without making mistakes.

## Features
* Random word generation based on difficulty level.
* Real-time countdown timer. 
* Level progression system.
* Increasing difficulty every 5 levels. 
* Immediate feedback (correct / incorrect/ Time Over). 
* Game over summary with stats. 
* Restart game functionality.

## How To Play
1. Click "Start Game"
2. A random word will appear on the screen 
3. Type the word exactly as shown 
4. Press Enter to submit your answer 
5. If correct → you advance to the next level 
6. If incorrect or time runs out → game over
